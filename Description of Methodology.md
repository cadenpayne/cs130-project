## Assumptions and Limitations
Assumptions and Limitations have an impact on data collection. It allows there to be some holes in research and not everything can be read and believed fully. Some assumptions and limitations I experienced are:

### Assumptions
* I assumed a lot for each state that there would be a few crimes not fully shown due to privacy issues and other felons not being found during those years.
* Also, I assumed that since the crime is grouped into categories, that crime types are standardized not arbitrary. Like Property Crimes include burglary and larceny but there are many other kinds aren't explicitly said.
### Limitations
* Similar to above, only reported crimes are included. Some crimes like domestic violence may never be reported.
* The data set combines crime information for an entire state into one row per year. This means important local differences are hidden. So, states with big cities would have a bigger impact to states with smaller cities and populations.
